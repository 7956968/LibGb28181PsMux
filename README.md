# LibGb28181PsMux
PsMux for China GB28181
========================================================================
项目概述
========================================================================

PS打包在实际项目中很难直接调试,而SIP很容易抓包调试,实际中经常有人问PS如何打包,而各自的实现也不尽相同,在对接时经常会遇到困扰,我们将PS打包部分独立出来以求更多的人完善。

例子中有VLC播放的SDP。打开本机的7000端口, 播放RTP payload 96的PS流。

海康，大华，东方网力等大部分平台都测试过。
